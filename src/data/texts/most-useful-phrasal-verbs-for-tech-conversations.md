| Phrasal Verb    | Meaning (in tech context)                  | Examples (EN → ES)                                                                                                                                                                      |
| --------------- | ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **set up**      | Configure or initialize something          | 1. I need to set up the project locally. → Necesito configurar el proyecto localmente. <br> 2. She set up the database yesterday. → Ella configuró la base de datos ayer.               |
| **build on**    | Extend or improve something existing       | 1. We can build on this feature later. → Podemos mejorar esta funcionalidad después. <br> 2. This version builds on the previous one. → Esta versión se basa en la anterior.            |
| **fix up**      | Repair or improve (often quickly)          | 1. I’ll fix up the UI before release. → Arreglaré la interfaz antes del lanzamiento. <br> 2. He fixed up the broken styles. → Él arregló los estilos rotos.                             |
| **roll out**    | Release/deploy something to users          | 1. We will roll out the feature next week. → Lanzaremos la funcionalidad la próxima semana. <br> 2. The update was rolled out تدريجيًا. → La actualización fue desplegada gradualmente. |
| **push out**    | Release or send something (often quickly)  | 1. Let’s push out a hotfix. → Saquemos un hotfix rápidamente. <br> 2. They pushed out a new version. → Ellos sacaron una nueva versión.                                                 |
| **pull in**     | Bring something (data/code) into a project | 1. We pull in data from an external API. → Traemos datos de una API externa. <br> 2. This library pulls in many dependencies. → Esta librería trae muchas dependencias.                 |
| **hook up**     | Connect systems/components                 | 1. We need to hook up the frontend to the backend. → Necesitamos conectar el frontend con el backend. <br> 2. He hooked up the payment system. → Él conectó el sistema de pagos.        |
| **spin up**     | Quickly start a service/server             | 1. I’ll spin up a test server. → Voy a levantar un servidor de prueba. <br> 2. We spun up a new instance. → Levantamos una nueva instancia.                                             |
| **shut down**   | Stop a system/service                      | 1. The server was shut down for maintenance. → El servidor fue apagado por mantenimiento. <br> 2. We need to shut down unused services. → Necesitamos apagar servicios no usados.       |
| **break down**  | Stop working / fail                        | 1. The app breaks down under heavy load. → La app falla bajo mucha carga. <br> 2. The process broke down unexpectedly. → El proceso falló inesperadamente.                              |
| **figure out**  | Understand or solve something              | 1. I’m trying to figure out this bug. → Estoy tratando de entender este bug. <br> 2. We figured out the issue. → Descubrimos el problema.                                               |
| **track down**  | Find the cause of a problem                | 1. I tracked down the memory leak. → Encontré la fuga de memoria. <br> 2. They are tracking down the error. → Están buscando la causa del error.                                        |
| **clean up**    | Improve code / remove unnecessary parts    | 1. I need to clean up this code. → Necesito limpiar este código. <br> 2. She cleaned up unused files. → Ella eliminó archivos no usados.                                                |
| **comment out** | Temporarily disable code                   | 1. Comment out that line for testing. → Comenta esa línea para probar. <br> 2. I commented out the function. → Comenté la función.                                                      |
| **plug in**     | Add/connect a module or tool               | 1. You can plug in this library easily. → Puedes integrar esta librería fácilmente. <br> 2. The plugin was plugged in yesterday. → El plugin fue integrado ayer.                        |
