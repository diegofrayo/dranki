# Keep vs Stay

**`Keep`** and **`Stay`** can sometimes look similar because both can express continuity, but they are used differently.

| Word     | Main Idea                                                       | Common Structure                                            |
| -------- | --------------------------------------------------------------- | ----------------------------------------------------------- |
| **Keep** | Continue doing something / maintain a state / retain possession | `keep + verb(-ing)` / `keep + adjective` / `keep + noun`    |
| **Stay** | Remain in a place, condition, or situation                      | `stay + adjective` / `stay + place` / `stay + with someone` |

## Quick Comparison

| Sentence                  | Meaning                         |
| ------------------------- | ------------------------------- |
| *Keep studying English.*  | Continue the action of studying |
| *Stay focused.*           | Remain in the focused state     |
| *Keep the window closed.* | Maintain the window closed      |
| *Stay inside the house.*  | Remain in that location         |

## Easy rule:

* **Keep → continue / maintain / retain**
* **Stay → remain / do not leave / continue in a state**

---

## 1. **Keep = Continue doing something**

Use **keep + verb (-ing)** when an action continues.

**Structure:**

```text
keep + verb-ing
```

Examples:

* *Keep practicing English every day.*
  → Continúa practicando inglés todos los días.

* *She kept talking during the movie.*
  → Ella siguió hablando durante la película.

---

## 2. **Keep = Maintain a condition**

Use **keep + adjective** when something remains in a certain state.

**Structure:**

```text
keep + adjective
```

Examples:

* *Keep calm during the interview.*
  → Mantente tranquilo durante la entrevista.

* *Please keep the door open.*
  → Por favor mantén la puerta abierta.

---

## 3. **Keep = Retain / Not lose / Continue having**

Use **keep + noun** when talking about possession or retaining something.

**Structure:**

```text
keep + noun
```

Examples:

* *You can keep this book.*
  → Puedes quedarte con este libro.

* *I keep my notes in a folder.*
  → Guardo mis notas en una carpeta.

---

## 4. **Stay = Remain in a place**

Use **stay** when talking about location.

**Structure:**

```text
stay + place
```

Examples:

* *We stayed at home all day.*
  → Nos quedamos en casa todo el día.

* *She is staying at a hotel.*
  → Ella se está hospedando en un hotel.

---

## 5. **Stay = Remain in a condition**

Use **stay + adjective** for physical, emotional, or situational states.

**Structure:**

```text
stay + adjective
```

Examples:

* *Stay focused on your goals.*
  → Mantente enfocado en tus metas.

* *Try to stay positive.*
  → Intenta mantenerte positivo.

---

## 6. **Stay = Continue being with someone / somewhere**

Use it when remaining with people or in a situation.

**Structure:**

```text
stay + with + person
```

Examples:

* *I stayed with my grandparents last weekend.*
  → Me quedé con mis abuelos el fin de semana pasado.

* *Can you stay with me for a while?*
  → ¿Puedes quedarte conmigo un rato?
