# Conditional sentences

Here’s a clear and practical table with the main types of conditional sentences in English:

| Conditional Type   | Structure (Brief Explanation)                                                                                       | Examples (EN → ES)                                                                                                                                                                                                                                                   |
| ------------------ | ------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Zero Conditional   | If + present simple, present simple. Used for general truths or facts.                                              | 1. If you heat water to 100°C, it boils. → Si calientas agua a 100°C, hierve. <br> 2. If it rains, the ground gets wet. → Si llueve, el suelo se moja.                                                                                                               |
| First Conditional  | If + present simple, will + base verb. Used for real and possible future situations.                                | 1. If I study, I will pass the exam. → Si estudio, aprobaré el examen. <br> 2. If it rains tomorrow, we will stay home. → Si llueve mañana, nos quedaremos en casa.                                                                                                  |
| Second Conditional | If + past simple, would + base verb. Used for hypothetical or unlikely situations in the present/future.            | 1. If I had more time, I would learn another language. → Si tuviera más tiempo, aprendería otro idioma. <br> 2. If I were you, I would take that job. → Si yo fuera tú, tomaría ese trabajo.                                                                         |
| Third Conditional  | If + past perfect, would have + past participle. Used for unreal situations in the past.                            | 1. If I had studied, I would have passed the exam. → Si hubiera estudiado, habría aprobado el examen. <br> 2. If they had left earlier, they would have arrived on time. → Si hubieran salido antes, habrían llegado a tiempo.                                       |
| Mixed Conditional  | Combination of second and third conditionals. Used to connect past conditions with present results (or vice versa). | 1. If I had studied engineering, I would have a better job now. → Si hubiera estudiado ingeniería, tendría un mejor trabajo ahora. <br> 2. If I were more disciplined, I would have finished the project. → Si fuera más disciplinado, habría terminado el proyecto. |
