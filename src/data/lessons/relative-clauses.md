# Relative Clauses

A relative clause is a part of a sentence that gives extra information about a noun. It usually starts with words like:

- who (people)
- which (things)
- that (people or things)
- where (places)
- whose (possession)

👉 Example:

- The developer who built this app is very skilled. → `El desarrollador que construyó esta aplicación es muy hábil.`

## Types of Relative Clauses

### 1. Defining Relative Clauses (Essential information)

These clauses are necessary to understand which person or thing we are talking about.

❗ No commas are used.

| Type                          | Examples (EN → ES)                                                                                                                                                              |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Defining Relative Clauses** | 1. The book that I bought is very interesting. → El libro que compré es muy interesante. <br> 2. The person who called you is my friend. → La persona que te llamó es mi amiga. |

### 2. Non-Defining Relative Clauses (Extra information)

These clauses add extra (non-essential) information.

❗ Commas are required.

❗ You cannot use that here.

| Type                              | Examples (EN → ES)                                                                                                                                                                                              |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Non-Defining Relative Clauses** | 1. My laptop, which I bought last year, is very fast. → Mi portátil, que compré el año pasado, es muy rápido. <br> 2. John, who works with me, is very talented. → John, que trabaja conmigo, es muy talentoso. |

### 3. Relative Clauses with “whose” (Possession)

Used to show that something belongs to someone.

| Type                   | Examples (EN → ES)                                                                                                                                                                                                         |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Whose (possession)** | 1. The developer whose code is clean works here. → El desarrollador cuyo código es limpio trabaja aquí. <br> 2. The company whose product we use is growing fast. → La empresa cuyo producto usamos está creciendo rápido. |

### 4. Relative Clauses with “where” (Place)

Used to refer to places.

| Type                   | Examples (EN → ES)                                                                                                                                                                                                         |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Whose (possession)** | 1. The developer whose code is clean works here. → El desarrollador cuyo código es limpio trabaja aquí. <br> 2. The company whose product we use is growing fast. → La empresa cuyo producto usamos está creciendo rápido. |

### 5. Relative Clauses with “when” (Time)

Used to refer to time.

| Type            | Examples (EN → ES)                                                                                                                                                       |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **When (time)** | 1. I remember the day when we met. → Recuerdo el día cuando nos conocimos. <br> 2. That was the moment when everything changed. → Ese fue el momento cuando todo cambió. |

## Quick Summary

- Defining → essential info (no commas)
- Non-defining → extra info (with commas)
- whose → possession
- where → place
- when → time
